# MuseBid Presentation Poster - Instructions

## Overview
A professional academic poster has been created for your Art Auction Website presentation. The poster includes all the required sections you mentioned.

## File Location
`presentation_poster.html`

## How to Use

### 1. View the Poster
Simply open `presentation_poster.html` in any web browser:
- Double-click the file, or
- Right-click and select "Open with" â†’ Your preferred browser

### 2. Customize the Poster

#### Add Your Website Screenshot
1. Take a screenshot of your MuseBid homepage when running
2. Save it as `homepage_screenshot.png` in the `static/images/` folder
3. In `presentation_poster.html`, find line with the placeholder and replace:
   ```html
   <div class="website-preview-placeholder">...</div>
   ```
   with:
   ```html
   <img src="static/images/homepage_screenshot.png" alt="MuseBid Homepage">
   ```

#### Add Your Personal Information
Find the "Student Information" section in the footer and replace:
- `[Your Full Name]` with your actual name
- `[Your Student ID]` with your student ID
- `[your.email@university.edu]` with your email
- `[Your Program/Major]` with your program
- `[Your University Name]` with your institution
- `[2024/2025]` with the correct academic year

#### Customize Acknowledgement (Optional)
Update the acknowledgement section with specific names of your supervisor, instructor, or anyone you'd like to thank.

### 3. Print the Poster

#### For Digital Presentation
- Use the HTML file directly in your presentation
- It's responsive and will look good on any screen

#### For Physical Printing
1. Open the file in your browser
2. Press `Ctrl + P` (Windows/Linux) or `Cmd + P` (Mac)
3. Select these print settings:
   - **Layout:** Landscape (recommended) or Portrait
   - **Paper Size:** A3 or A2 for best results
   - **Margins:** Minimal or None
   - **Background Graphics:** Enabled/On (important for colors)
4. Print to PDF first to preview before physical printing

### 4. Export as PDF
1. Open in browser
2. Print (Ctrl+P / Cmd+P)
3. Select "Save as PDF" as printer
4. Save the file
5. This PDF can be:
   - Emailed to your instructor
   - Printed at a print shop
   - Shared digitally

## Design Features

- **Professional Color Scheme:** Browns and golds matching your website theme
- **Organized Layout:** Clear sections with visual hierarchy
- **Print-Ready:** Optimized for both digital and physical presentation
- **Responsive:** Works on different screen sizes
- **Easy to Read:** Clear fonts and good contrast

## Tips for Presentation

1. **Practice Your Pitch:** Be ready to explain each section
2. **Know Your Code:** Be familiar with how you implemented each function
3. **Demo Ready:** Have your website running for live demonstration
4. **Print Quality:** For physical posters, use high-quality paper (200gsm+)
5. **Size Recommendation:** A2 (420 x 594 mm) or A3 (297 x 420 mm) for good visibility

## Troubleshooting

**Colors not showing when printing?**
- Enable "Background Graphics" in print settings

**Layout looks wrong?**
- Try a different browser (Chrome recommended)
- Check your screen resolution

**Text too small/large?**
- Use browser zoom (Ctrl + or Ctrl -)
- Adjust print scale in print dialog

## Need Changes?
